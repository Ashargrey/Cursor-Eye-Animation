# 👀 Cursor Eye Animation

An interactive and fun web project where the eyes follow your cursor movement in real-time.

## 🔗 Live Demo

https://ashargrey.github.io/Cursor-Eye-Animation/

## 💡 Features

* Real-time eye tracking
* Smooth cursor-follow animation
* Responsive design
* Clean and minimal UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ⚙️ How It Works

This project uses:

* `getBoundingClientRect()` to get the position of the eyes
* `Math.atan2()` to calculate the angle between the cursor and eye center
* Mouse movement events to update eye direction dynamically
